# Growvy AI Internship - Image Classifier

**Project Documentation**

**Position**: Artificial Intelligence (AI) 

**Internship Project Title**: Optimized Image Classifier Deployment   

1. **Executive Summary**:
The objective of this 6-week project was to build, optimize, and deploy a robust AI solution. I developed an image classification model using the CIFAR-10 dataset, capable of identifying objects across 10 distinct categories.

2. **Development Phases**:
* Data Preprocessing (Week 4): Images were normalized to a range of 0–1 to ensure faster convergence during training.

* Model Architecture (Week 4): I implemented a Convolutional Neural Network (CNN) using Keras to extract spatial features from the image data.

* Optimization (Week 5): To improve accuracy and prevent overfitting, I applied Dropout layers, Batch Normalization, and Data Augmentation techniques.

* Deployment (Week 6): The final model was integrated into a Flask API, allowing for real-time predictions via HTTP POST requests.

3. **Performance & Results**:
The model achieved a successful classification during testing (e.g., identifying a dog with class_id: 5), demonstrating its readiness for deployment.
